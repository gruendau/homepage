## Enumerierungen

Eine [Enumeration](https://docs.swift.org/swift-book/documentation/the-swift-programming-language/enumerations) <_enumeration_> ist eine Aufzählung von Werten. Für die Deklarierung wird das Schlüsselwort `enum` genutzt.  

```swift
enum Namen {
    case volker
    case nils
}
